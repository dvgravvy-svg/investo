Idea: An investing app that teaches you how to trade and invest starting from the fundamentals to more advanced skills in trading.

Problem: Many people feel overwhelmed and don’t know how to start trading. They also feel very demotivated when they are unable to learn effectively and just give up on trading easily

Name: INVESTO

Key functions:
1. Guided lessons
- Starts with basics.
- Breaks down complex financial concepts into bite-sized, easy-to-understand units.
- Tests knowledge with quick knowledge checks before moving to the next level.

2. Paper trading simulator
- Allows hands on learning on how trading actually works.
- Immersifies the experience of actual trading without the use of real money.
- Helps users to gain more experience and confidence on trading/ investing.

3. AI chatbot tutor
- User can ask questions regarding a certain lesson or what they don’t understand on how trading actually works.
- Provides analysis on charts so that the user can understand how do they read/ analyse a chart.

4. Gamified learning platform 
- Makes the learning experience more fun and interactive, ensuring the user doesn’t feel bored and unable to learn effectively.
- Provides immediate AI feedback on answers.
